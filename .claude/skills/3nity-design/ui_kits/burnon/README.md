# BurnOn · 3NITY UI Kit

BurnOn is the AI-powered fitness app referenced in the brand book. Mobile, dark theme, 3NITY visual grammar applied to a product surface:
- Black fond + lime accents
- Pixel isotype + monospace metadata
- No rounded corners (pixel grammar)
- VT323 used for stats / score readouts only

Components:
- `StatusBar.jsx` — iOS-style status bar treated as part of the design
- `Header.jsx` — page header with isotype + meta tag
- `Stat.jsx` — large numeric stat with VT323 readout
- `WorkoutCard.jsx` — list card for workouts
- `TabBar.jsx` — bottom navigation
- `Screens.jsx` — 4 stitched screens
