# Posters · 3NITY UI Kit

The brand's primary surface. 3NITY communicates through editorial posters that mix:
1. **Lime fond + black tetris invasion** (energy)
2. **Cream + black grafismo** (gallery / editorial)
3. **Black fond + neon glow** (lab / nighttime)

`index.html` lets you flip between three composed posters. Each is built from the same atomic React parts so they can be re-assembled.

Components:
- `MetaTag.jsx` — corner metadata blocks (`HERO SHOT / EDITORIAL`, `3NITY / 09°`)
- `Isotype.jsx` — the pixel cruciform, sized + colored
- `Wordmark.jsx` — `3nity` lockup
- `TetrisField.jsx` — 8×6 grid of pixel blocks invading from a corner
- `RadialSun.jsx` — the lime radial gradient sun
- `CircleGlyph.jsx` — solid / striped / cross visual punctuation
- `PosterFrame.jsx` — outer frame for an A2-style billboard
